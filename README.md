# Morpheus modernization archive

> [!IMPORTANT]
> This repository is preserved as historical source for the Morpheus modernization project.
>
> The former `soapbubble.online` domain and its subdomains are no longer controlled by or affiliated with Soap Bubble Productions. Do not rely on content or services published there.
>
> Historical browser, desktop, and mobile releases depended on backend and cloud services that are no longer available. There is currently no supported playable release.

Morpheus was a multi-year modernization of Soap Bubble Productions’ 1998 graphical adventure game. The surviving repository preserves the data-driven game runtime, web application, cross-platform packaging, infrastructure, tags, and development history.

- [Project history and current status](https://www.flick.ing/~/projects#morpheus)
- [Historical itch.io release page](https://soapbubble.itch.io/morpheus)

Some archived configuration files still contain references to retired infrastructure. They are historical implementation records, not active or trusted endpoints.
